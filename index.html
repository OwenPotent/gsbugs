➡️ Website not loading / responding

The website's slowed due to images loading more often per user, a total of 64 request/users visit the website every second, during the opening I could tell the website crashed since it took me around 3 seconds for the server to load clearly, 3 seconds is a lot, and the users are making a traffic out of it.

➡️ Login Page

The login page was not supposed to be visible to customers, its purpose is meant for users who are chosen by Wolfie.

➡️ Website Layout Not Working Properly

When you load the website, you might encounter buggy content such as overlayed images, weird navigation bar, bigger images, search bar and so on. This is due to the website having problems responding to certain devices.

➡️ Some Pages Showing Up On Search

As you search using the "search bar" feature, you might encounter some pages that weren't meant to be shown, e.g. Privacy Policy, FAQ, Contact Us.

➡️ Dummy Text On Asset Page

When loading an asset page, you may find "Lorem Ipsum", or dummy texts on the description and others.

➡️ Billing Details

At the moment, when trying to download an asset, you are prompt to enter your name and phone number. This should not happen but will be fixed soon.
